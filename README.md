# KMeans Clustering - Streamlit App

This project performs unsupervised clustering on customer data using the Elbow Method and KMeans with PCA dimensionality reduction. Built with Streamlit and modularized in Python.

## 📁 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🧠 Features
- Upload dataset (mall_customers.csv)
- Preprocessing and PCA
- Elbow plot to determine optimal clusters
- Train model with k=6
- Cluster visualization (optional)

## 📦 Deployment
Ready to deploy on [Streamlit Cloud](https://streamlit.io/cloud).
